# Nexora

### Discover Your Next Opportunity

Nexora is a global opportunity platform that helps students discover:

- Scholarships
- Competitions
- Hackathons
- Olympiads
- Research Programs
- Internships
- Conferences
- Exchange Programs
- Summer Schools
- Fellowships
